How to use:

  • User API token - Your assigned token. If no token is specified, a test token will be used.
  • Collections Table - List of collections within the API
  • Collection Update Button - This button will bring up fields for updating the collection and associated items.
  • Add Collection Items - Place a list of Record Identifiers to add items to a collection. Identifiers should be entered one per line.
